## 📊 Project Description

This is a console-based **Sorting Algorithm Visualizer** developed in C++ as part of the CodTech Internship (Task CITS2826). The application demonstrates the working of various sorting algorithms through step-by-step visualization in the terminal, helping users understand how data is sorted and how different algorithms compare in terms of execution and behavior.

## ✨ Features

* 🫧 Bubble Sort Visualization
* 🎯 Selection Sort Visualization
* 📥 Insertion Sort Visualization
* ⚡ Quick Sort Visualization
* 🔁 Generate Random Arrays
* 📊 Real-time Console-Based Visualization
* 🖥️ User-Friendly Menu Driven Interface

## 🛠️ Technologies Used

* **Programming Language:** C++
* **Concepts:** Object-Oriented Programming (OOP)
* **Data Structures:** STL Vector
* **Algorithms:** Bubble Sort, Selection Sort, Insertion Sort, Quick Sort
* **Libraries:** `<vector>`, `<thread>`, `<chrono>`, `<algorithm>`


